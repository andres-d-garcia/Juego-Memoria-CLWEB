# 🐉 Memory Tenkaichi 🐉

¡Despierta tu ki y pon a prueba tu memoria en **Memory Tenkaichi**! 
Este es un emocionante juego de cartas emparejadas (Memory Match) inspirado en el vasto universo de Dragon Ball. Los jugadores pueden desafiar su mente intentando encontrar todos los pares ocultos antes de que el tiempo se agote o compitiendo contra un amigo para ver quién es el guerrero más fuerte.

## 🚀 Instrucciones de Ejecución

Este proyecto está construido con tecnologías web estándar (HTML5, CSS3 y JavaScript Vanilla), por lo que no requiere la instalación de complejas dependencias ni bases de datos.

### Opción 1: Ejecución local rápida
1. Clona este repositorio en tu computadora:
   ```bash
   git clone https://github.com/tu-usuario/Juego-Memoria-CLWEB.git
   ```
2. Navega hasta la carpeta del proyecto y entra al directorio `docs`.
3. Haz doble clic en el archivo `index.html` para abrirlo directamente en tu navegador web predeterminado.

### Opción 2: Ejecución mediante Live Server (Recomendado)
Para evitar restricciones de seguridad del navegador relacionadas con los módulos de JavaScript (CORS):
1. Abre la carpeta del proyecto en **Visual Studio Code**.
2. Instala la extensión **Live Server**.
3. Haz clic derecho sobre el archivo `docs/index.html` y selecciona **"Open with Live Server"**.

### Opción 3: Jugar en línea
¡El juego está desplegado en GitHub Pages! Puedes jugarlo ahora mismo haciendo clic en el siguiente enlace:
👉 **Jugar Memory Tenkaichi** *(https://andres-d-garcia.github.io/Memory-Tenkaichi/)*.

## 🎨 Temáticas Implementadas

El juego cuenta con tres mazos de cartas únicos, diseñados con artes vibrantes y fieles a la serie:

1. 🥋 **Guerreros Z:** Encuentra a los héroes que protegen la Tierra, incluyendo a Goku, Vegeta, Gohan, Piccolo y muchos más.
2. 🔮 **Misceláneos:** Artículos icónicos del mundo de Dragon Ball, como las Semillas del Ermitaño, el Radar del Dragón, las Cápsulas Hoi-Poi y, por supuesto, las Esferas del Dragón.
3. 😈 **Villanos:** Enfréntate a la memoria de los antagonistas más temibles, desde Freezer y Cell hasta Majin Buu y Goku Black.

## 🏆 Logros y Desafíos

Para hacer la experiencia más inmersiva, hemos implementado un sistema de **Logros de Sesión**. Estos aparecerán como notificaciones flotantes durante el juego cuando cumplas ciertas condiciones:

| Icono | Título | Descripción | Cómo Desbloquearlo |
| :---: | :--- | :--- | :--- |
| 🥇 | **Primer paso** | Encontrar el primer par. | Haz tu primer emparejamiento exitoso en cualquier partida. |
| 🔥 | **Racha caliente** | 3 pares consecutivos sin fallar. | Encuentra 3 pares de cartas seguidos sin equivocarte en los intentos intermedios. |
| ⚡ | **Velocista** | Completar en Fácil en < 30s. | Supera un tablero de 16 cartas (Fácil) en menos de 30 segundos. |
| 🎯 | **Sin titubeos** | Par en el primer intento. | Acierta un par de cartas en el mismísimo primer movimiento del tablero. |
